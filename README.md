# LLM RAG AI Assistant

This project implements a Retrieval Augmented Generation (RAG) AI assistant capable of answering questions from custom documents.

## Features

- Document ingestion
- Vector embeddings using FAISS
- Semantic search
- LLM powered question answering
- FastAPI backend

## Tech Stack

- Python
- LangChain
- OpenAI API
- FAISS
- FastAPI

## Use Cases

- AI document assistant
- Knowledge base chatbot
- Intelligent search system
